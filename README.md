# n8n Automation Builder

A comprehensive n8n workflow development environment with integrated tools, skills, and reference materials for building production-grade automations.

## Quick Start

```bash
# 1. Clone the main project
git clone https://github.com/pranav-pk1917/n8n_Automation_Builder.git

# 2. Enter the project directory
cd n8n_Automation_Builder

# 3. Clone the reference library (7000+ n8n workflow examples)
git clone https://github.com/pranav-pk1917/n8n_Automation_Builder_ReferenceLibrary.git Reference_Library

# 4. Open in Cursor IDE
cursor .
```
